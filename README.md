## 🤖 AI 작업 지식 저장소 (Streamlit + G-Drive)

### 1. 프로그램 목적
* **영구 보관:** 접속할 때마다 초기화되는 Streamlit의 한계를 극복하고 데이터를 구글 드라이브에 영구 저장합니다. (streamlit.io 사이트 - txt, json, 등의 자료 파일은 휘발성으로 자동 삭제됨 ) 
* **자산화:** AI가 생성한 마크다운(Markdown) 답변을 체계적으로 수집하여 나중에 **NotebookLM**이나 **Gemini 확장 기능**에서 나만의 데이터베이스로 활용합니다.
* **통합 관리:** 직접 쓴 글뿐만 아니라 내 컴퓨터에 있는 기존 파일도 한곳에 모읍니다.

### 2. 주요 기능
* **📑 목록 조회:** 구글 드라이브 .
* **✍️ 직접 저장 (Write):** AI의 답변을 복사해서 붙여넣고 파일명을 지정하면 즉시 `.md` 파일로 생성됩니다.
* **📁 파일 업로드 (Upload):** 내 컴퓨터의 로컬 파일(MD, TXT, PDF 등)을 브라우저에서 선택해 드라이브로 전송합니다.

---

### 3. 시스템 구조 (Data Flow)



1.  **사용자 인터페이스 (Streamlit):** 웹 브라우저를 통해 텍스트 입력 및 파일 선택.
2.  **인증 레이어 (Service Account):** PC방식과, 웹방식의 인증이 다름 (구글콘솔 참조) 
3.  **저장소 (Google Drive)**
4.  **활용 단계 (AI Agent):** 저장된 데이터를 Gemini가 학습하여 사용자 맞춤형 답변 제공.

---

### 4. 사용 전 필수 체크리스트
* **API 활성화:** Google Cloud Console에서 *
* streamlit.io사이트에서 credential 관리 방법 참조
* **권한 공유:** 서비스 계정 이메일(`...iam.gserviceaccount.com`)에 드라이브 폴더의 **'편집자'** 권한을 부여해야 합니다.

---

### 5. 기대 효과
